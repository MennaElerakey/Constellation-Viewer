Constellation Viewer is a beginner-friendly Python program that lets users explore and learn about some of the most famous constellations in the night sky. It combines visual ASCII art representations with concise scientific information, making it both educational and fun!

Features

View 10 popular constellations: Orion, Ursa Major, Cassiopeia, Scorpius, Leo, Cygnus, Taurus, Crux, Pegasus, and Andromeda.

Enjoy ASCII art illustrations of each constellation.

Learn interesting scientific facts, including notable stars, nebulae, and historical significance.

Interactive experience: choose constellations, view information, and continue exploring as long as you like.

How to Use

Run the Python program.

Enter your name to personalize your experience.

Choose a constellation from the list.

Read about its stars, nebulae, and interesting facts.

Optionally, explore other constellations or exit the program.
